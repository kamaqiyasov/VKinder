# VKinder - Бот для знакомств ВКонтакте

## Описание
Бот для поиска людей для знакомств в социальной сети ВКонтакте...

## Установка
1. Клонировать репозиторий
2. Установить зависимости: `pip install -r requirements.txt`
3. Настроить .env файл
4. Создать базу данных
5. Запустить: `python src/main.py`

## Настройка
1. Создать группу ВКонтакте
2. Получить групповой и пользовательский токены
3. Настроить PostgreSQL
